# Devise Example for Rails 3

Example App using devise. http://github.com/plataformatec/devise

## Installation

Install bundler if you haven't yet:

```
gem install bundler
```

Install the gems:

```
bundle install
```

This will install Rails 3, neo4j and Devise.
Start the server:

  rails s
